# Memory Games - Jewish Knowledge Quizzes

A suite of interactive browser-based memory games designed to test and improve knowledge of Jewish texts. This project features a central landing page and multiple timed quizzes with "Sporcle-style" list inputs.

### 1. Mishna Quiz
* **Goal:** Name all 63 Tractates (Masechtos) of the Mishna.
* **Time Limit:** 6 minutes, 13 seconds.
* **Features:** Organize by Seder (Order).

### 2. Parsha Quiz
* **Goal:** Name all 54 Parshas of the Torah.
* **Time Limit:** 6 minutes, 13 seconds.
* **Features:** Organized by Chumash (Book).

## âœ¨ Key Features

* **Bilingual Support:**
    * The board displays names in **Hebrew**.
    * The game accepts input in **English** (multiple spellings) or **Hebrew**.
* **Smart Input:**
    * Case-insensitive.
    * Handles variations (e.g., "Shabbos" vs "Shabbat").
    * Automatic duplicate detection.
* **Game Controls:**
    * **Start Button:** Initiates the session and enables text entry to ensure accurate timing.
    * **Give Up:** Reveals all missing answers in red.
    * **Play Again:** Resets the board without refreshing the page.
* **Responsive Grid:** Layout automatically adjusts to show columns clearly.
